# User

```
type User {
    id: ID!
    username: String!
    email: String!
    password: String!
    created_at: Date! default=Date.now()
    updated_at: Date! default=Date.now()
    bio: String
    image: String
    email_verified: Boolean
    refresh_token: String default=''
    access_token: String default=''
}
```
